# Bocce Simulation System

This repository contains a simulation system for the game of Bocce. The project aims to create a realistic simulation of Bocce gameplay, allowing for analysis and exploration of various game strategies.

## Installation

To get started, clone the repository and install the required packages.

```bash
# Clone the repository
git clone https://github.com/francedap/simulazione-bocce.git

# Change directory to the project
cd simulazione-bocce

# Install dependencies
pip install -r requirements.txt
```

## Usage

To run the simulation, execute the main Python file:

```bash
python main.py
```
